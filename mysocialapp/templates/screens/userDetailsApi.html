EndPoints

  • GET http://127.0.0.1:8080/api/users - returns all the registered users and their details
    http://127.0.0.1:8080/api/users

  • GET http://127.0.0.1:8080/api/appUserDetail/[APPUSER ID] - return the appuser details and all we know about it
    http://127.0.0.1:8080/api/appUserDetail/{{appuser.id}}

  • GET http://127.0.0.1:8080/api/userPostList/[APPUSER ID] - returns all the posts of the user
    http://127.0.0.1:8080/api/userPostList/{{appuser.id}}

  • GET http://127.0.0.1:8080/api/userFriendList/[APPUSER ID] - return a list of all friends/followings and their details of the user
    http://127.0.0.1:8080/api/userFriendList/{{appuser.id}}

  • GET http://127.0.0.1:8080/api/userStatus/[APPUSER ID] - return the status details of the user
    http://127.0.0.1:8080/api/userStatus/{{appuser.id}}
